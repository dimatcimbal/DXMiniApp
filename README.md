# DXMiniApp

Lightweight DirectX setup for graphics programming experiments.

### Project Structure

```textmate
DXMiniApp/
├── build/             # Build output directory (generated by CMake)
├── src/               # The root source directory
│   ├── main.cpp       # Main application entry point
│   ├── Win32Application.h    # Header for the Win32 application class
│   ├── Win32Application.cpp  # Implementation for the Win32 application class
│   │
│   ├── Util/          # Utility classes
│   │   ├── BaseFileProvider.h # Base class for file providers
│   │   ├── BaseFileProvider.cpp # Base class implementation (if any)
│   │   ├── WorkingDirFileProvider.h # File provider for working directory
│   │   └── WorkingDirFileProvider.cpp # Implementation for working directory file provider
│   │
│   └── Window/        # App Window and Views
│       ├── BaseView.h          # Base class for all views
│       ├── MainWindow.h        # Header for the main application window
│       ├── MainWindow.cpp      # Implementation for the main application window
│       ├── SceneView.h         # Header for the Scene rendering view
│       ├── SceneView.cpp       # Implementation for the Scene rendering view
│       ├── FileView.h          # Header for the file list view
│       ├── FileView.cpp        # Implementation for the file list view
│       ├── SceneTree.h         # Header for the scene tree view (previously NodeView)
│       └── SceneTree.cpp       # Implementation for the scene tree view (previously NodeView)
│
├── .clang-format      # ClangFormat configuration for code style
├── .gitignore         # Git ignore rules
├── app.rc             # Windows resource file
├── ci.yml             # GitHub Actions CI workflow
├── CMakeLists.txt     # CMake build configuration
├── DXMiniApp.manifest # Application manifest file
├── housekeeper.ps1    # Build automation script
├── README.md          # Project documentation
├── TODO.md            # Project to-do list
```

### Quick Start
```powershell
# Clone and build
git clone <repository-url>
cd DXMiniApp
.\housekeeper.ps1

# Or step by step
.\housekeeper.ps1 -Build
```
